# Backend Iniciante com Java

## Descrição

Primeiro contato com programação utilizando Java para criar um programa que recebe dados, toma decisões e mostra resultados no terminal.

---

## CBL

### Mentor Driven Challenge
Criar uma solução em Java.

### Big Idea
**Segurança Digital**

### Challenge
Detectar mensagens que sejam golpes digitais.

---

## Goals

- Traduzir uma lógica para uma linguagem de programação.
- Entender como o backend transforma dados em decisão.
- Trabalhar em grupo.
- Entender o recebimento de dados do usuário e processá-los.

---

## Grupos

- 2 a 3 estudantes por grupo.

---

## Entregáveis

- Programa em Java funcional no terminal.
- Repositório no GitHub.
- Uma regra de negócio implementada no programa.
- Pelo menos um cenário de teste.
- Explicação da lógica e da regra de negócio utilizada pelo grupo.

---

## Apresentação

- **5 minutos** por grupo. Apresentar para toda a turma.
- **5 minutos** para feedback.

---

## Constraints (Obrigatoriedades)

- Utilizar **Java** como linguagem de programação.
- Aplicação executada via terminal.
- Foco em lógica, entrada, processamento e saída.
- Solução simples, sem interface gráfica.

---

  - Fluxograma

- Exemplos de mensagens fraudulentas.
- Levantamento de sinais de risco.

#### Aula 02
**Investigate** + **Checkpoint 1**:
- Definição das regras da solução.
- **Milestone:** entrega do fluxograma.
